# Linux Troubleshooting Lab

A collection of real-world Linux break/fix scenarios built to develop 
sysadmin and IT support skills. Each scenario follows a structured 
incident report format: Symptom → Investigation → Root Cause → Fix.

## Scenarios

| # | Scenario | Tools Used |
|---|----------|------------|
| 01 | [Nginx Won't Start](labs/01-nginx-wont-start/README.md) | systemctl, journalctl, nginx -t |
| 02 | Port 80 Blocked | Nginx running but port 80 unreachable | `sed`, `grep`, `curl` | ✅ Complete |

## Why I Built This
Practicing real diagnostic thinking rather than just memorizing commands.
